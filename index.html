Face Recognition Attendance System

1. Start

The process begins when a user opens the system.

2. Is it a New User?

The system checks whether the user is already registered.

3. If Yes → Capture Photograph

The user's face photo is collected for future recognition.

4. Store Face Data

The captured face image is saved in the database or folder.

5. Recognize Face

The system scans and identifies the face using AI model.

6. Face Recognized?

If face matches → continue. If not → retry or exit.

7. Store Attendance

Registration number, date, and time are stored in a CSV file.

8. End

The attendance process ends successfully.
